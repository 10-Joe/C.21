Arrastra este botón a tu barra de marcadores

¿Cómo se usa?

  1. Mantén presionado el botón negro de arriba y arrástralo hacia tu barra de marcadores (la barra blanca debajo de donde escribes las direcciones web).
  2. Entra a ZonaProp y busca cualquier propiedad.
  3. Una vez que estés viendo la propiedad en ZonaProp, haz clic en el marcador que guardaste.
  4. ¡Magia! Se abrirá una nueva pestaña con tu Ficha Técnica lista para imprimir como PDF.

Truco: Si no ves la barra de marcadores, presiona Ctrl + Shift + B.
